If you want to install it directly from the GitHub repository:

.. code-block:: bash

    git clone https://github.com/Francesco-Zeno-Costanzo/datxpy.git
    cd datxpy
    pip install -e .

Make sure to also install the dependencies:

.. code-block:: bash

    pip install -r requirements.txt
